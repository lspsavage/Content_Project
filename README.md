# Project Content Showcase - Data Analyst Portfolio

Repository ini berisi source code untuk halaman detail proyek (Project Content) yang terhubung dengan [Portofolio Utama Saya](https://lspsavage.github.io/My_porto/page_project.html).

Tujuan dari repository ini adalah menyediakan template tampilan yang modern dan interaktif untuk mempresentasikan hasil analisis data, dashboard, dan insight dari berbagai proyek data analyst yang telah saya kerjakan.

Saat ini, contoh konten yang diimplementasikan dalam repository ini adalah studi kasus **Boston Housing Price Analysis**.

## 🎯 Tujuan

- Memberikan presentasi visual yang lebih menarik untuk proyek data dibandingkan sekadar file Jupyter Notebook atau PDF.
- Menampilkan ringkasan eksekutif, pertanyaan riset, dashboard interaktif (Power BI), dan kesimpulan dalam satu halaman web yang responsif.
- Meningkatkan pengalaman pengguna (User Experience) saat menelusuri detail proyek dari portofolio utama.

## ✨ Fitur Tampilan

Halaman ini dibangun menggunakan teknologi web modern untuk menciptakan pengalaman visual yang dinamis:

- **Interactive Background**: Animasi partikel yang responsif menggunakan `tsParticles`.
- **Modern UI Effects**: Efek _glow_ pada kursor dan kartu, serta desain _glassmorphism_ yang elegan.
- **Scroll Animations**: Elemen muncul secara halus saat di-scroll menggunakan `ScrollReveal`.
- **Responsive Design**: Tata letak yang menyesuaikan otomatis untuk Desktop, Tablet, dan Mobile menggunakan Tailwind CSS.
- **Navigation**: Navigasi yang intuitif termasuk menu hamburger untuk tampilan mobile.

## 🛠️ Teknologi Frontend

- **HTML5**
- **Tailwind CSS** (Styling framework)
- **JavaScript** (Logic & Interactivity)
- **Library**:
  - `tsParticles` (Background animation)
  - `ScrollReveal` (Scroll animation)

## 🚀 Cara Menjalankan (Development)

Jika Anda ingin mengembangkan atau memodifikasi tampilan ini secara lokal:

1.  **Clone repository:**
    ```bash
    git clone https://github.com/lspsavage/[nama-repo-anda].git
    ```
2.  **Install dependencies:**
    ```bash
    npm install
    ```
3.  **Jalankan Tailwind CSS (Watch Mode):**
    ```bash
    npm run watch
    ```
4.  **Buka `index.html`** di browser pilihan Anda.
